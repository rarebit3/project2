# Project2 - Wine Cellar
## Date: 3/28/2022
### By Trevor Bond
***
### ***Description***
#### An application for tracking and cellaring wine and spirits. The app will utilize a database on mongoDB. Once a bottle is opened the app will prompt the user to write a review.
